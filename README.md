# Tab Bar Navigation (Flutter)

A Flutter app that demonstrates bottom tab bar navigation.

## Run Steps

1. Install Flutter (stable) and verify setup:
   ```bash
   flutter doctor
   ```
2. Clone this repo and open it:
   ```bash
   git clone <your-repo-url>
   cd TabBarNavigation
   ```
3. Install dependencies:
   ```bash
   flutter pub get
   ```
4. Run on simulator/device:
   ```bash
   flutter run
   ```
